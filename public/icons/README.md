# Gamified Task Tracker Icons

These placeholder SVG icons are for PWA functionality.

## Generate Real PNG Icons

Use an online tool or ImageMagick to convert SVGs to PNGs:

```bash
# Using ImageMagick
convert icon-192.png -resize 192x192 icon-192.png
convert icon-512.png -resize 512x512 icon-512.png
```

Or use https://realfavicongenerator.net/ to generate a complete icon set.
